# ☕ Sistema de Reservas para Coworking — Java POO

Este repositório apresenta a implementação de um sistema de gerenciamento de coworking desenvolvido em Java, como parte das atividades da disciplina de Programação Orientada a Objetos (POO) do curso de Sistemas de Informação — Ifes Serra.

O projeto simula o funcionamento de um ambiente de coworking, permitindo o cadastro de clientes, gerenciamento de espaços e realização de reservas de salas e estações de trabalho.

## 🚀 Funcionalidades do Sistema

O sistema foi desenvolvido para possibilitar:

* Cadastro de clientes
* Cadastro de salas de reunião
* Cadastro de estações de trabalho
* Reservas por:

  * dia inteiro
  * turno
  * horário específico
* Consulta de reservas
* Filtro de reservas por cliente e por data
* Controle automático de disponibilidade
* Cálculo automático de valores das reservas

## 🧠 Conceitos de Programação Utilizados

Durante o desenvolvimento foram aplicados diversos conceitos importantes de orientação a objetos, incluindo:

* Herança
* Polimorfismo
* Sobrecarga de métodos
* Reescrita de métodos
* Encapsulamento
* Associação entre classes
* Manipulação de datas e horários
* Estruturas de dados com Array e ArrayList

## 📂 Estrutura do Projeto

O sistema é dividido em classes responsáveis por diferentes partes da aplicação, como:

* Clientes
* Reservas
* Espaços
* Salas
* Estações de trabalho
* Datas e horários
* Entrada e interação com usuário

## 💻 Tecnologias Utilizadas

* Java ☕
* Programação Orientada a Objetos
* UML
* IntelliJ IDEA / VS Code

## 🎯 Objetivo do Projeto

O objetivo principal deste trabalho é reforçar os conceitos de orientação a objetos através da construção de um sistema completo utilizando Java, aplicando organização, reutilização de código e modelagem de classes.
